# Security

## Reporting
For sensitive issues, contact the team via [X @normiedata](https://x.com/normiedata). Do not open public issues with exploit detail.

## Principles
- Least privilege for infrastructure credentials  
- No secrets in this public repository  
- Wallet-based contribution attribution  
- Fingerprinting before dataset admission  
- Privacy masking path for sensitive regions in media  

## Scope of this repo
This repository contains **documentation and brand assets only**.

It does **not** contain:
- private keys or seed phrases  
- API tokens / RPC keys  
- application source code  
- production database dumps
